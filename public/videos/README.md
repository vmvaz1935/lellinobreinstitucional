# Vídeo do Hero

Coloque o vídeo institucional nesta pasta com o nome:
- `hero-video.mp4` (formato MP4 - recomendado)
- `hero-video.webm` (formato WebM - opcional, para melhor compatibilidade)

## Especificações recomendadas:
- Formato: MP4 (H.264)
- Resolução: 1920x1080 ou superior
- Duração: 15-60 segundos (loop automático)
- Tamanho: Otimizado para web (máximo 10MB recomendado)

O vídeo será reproduzido automaticamente em loop, sem som, na seção hero do site.
